# SDL_bool

A boolean type: true or false.

## Header File

Defined in [<SDL3/SDL_stdinc.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_stdinc.h)

## Syntax

```c
typedef int SDL_bool;
```

## Version

This datatype is available since SDL 3.0.0.
