# SDL_HAPTIC_CONSTANT

Constant effect supported.

## Header File

Defined in [<SDL3/SDL_haptic.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_CONSTANT     (1u<<0)
```

## Remarks

Constant haptic effect.
